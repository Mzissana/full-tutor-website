# Форма обратной связи Cloudflare

1. Создайте Worker `mzissana-contact` в панели Cloudflare
2. Загрузите содержимое `src/index.js`
3. Добавьте Secrets: `TELEGRAM_BOT_TOKEN` и `TELEGRAM_CHAT_ID`
4. Скопируйте адрес опубликованного Worker в `public/contact-endpoint.js`

Секреты не добавляются в этот репозиторий и не должны передаваться в чат.
